# Development phases

This directory contains the authoritative development phase documentation for the AgenticVerdict multi-business-domain intelligence platform.

## Phase structure

### Core platform (`00-core`)

The core platform encompasses the foundational capabilities required to deliver business intelligence across multiple domains:

| Sub-phase                 | Focus                                         | Status      |
| ------------------------- | --------------------------------------------- | ----------- |
| `00-foundation`           | Infrastructure, multi-tenancy, core services  | Complete    |
| `01-connectors`           | Data connector integrations and normalization | In progress |
| `02-intelligence`         | AI agent orchestration and analysis           | Planned     |
| `03-insights`             | Report generation and delivery                | Planned     |
| `04-production-hardening` | Security, performance, production readiness   | Planned     |
