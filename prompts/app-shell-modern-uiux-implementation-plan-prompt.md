## Prompt: App Shell Modernization Analysis and Implementation Plan

You are a senior frontend architect and product UI/UX strategist.

### Context

- The current implementation at `/apps/frontend/src/components/layout/AppShellLayout.tsx` is a basic prototype shell.

### Objective

- Define a clear, professional path to evolve the current shell into a modern, production-grade experience with strong UI and UX quality.

### Task

Conduct a thorough analysis of the current implementation and produce a comprehensive implementation plan that upgrades the shell using modern, industry-aligned standards and best practices.

### Required Output

Create a structured planning document that includes:

1. **Current State Assessment**
   - Strengths, weaknesses, and UX/UI gaps in the existing shell.
   - Technical and design constraints discovered in the current implementation.
2. **Target Experience Definition**
   - Vision for the modern shell across web, desktop, and mobile form factors.
   - Core principles for usability, accessibility, responsiveness, and consistency.
3. **Architecture and Design Strategy**
   - Component structure recommendations and layout system approach.
   - Styling/theming, design token usage, and scalability considerations.
4. **UX Improvement Plan**
   - Navigation, information hierarchy, interaction patterns, and feedback states.
   - Accessibility expectations (WCAG 2.1 AA) and internationalization readiness (including RTL/LTR support).
5. **Implementation Roadmap**
   - Phased execution plan (short-term, mid-term, long-term).
   - Prioritized tasks, dependencies, risks, and mitigation strategies.
6. **Validation and Quality Gates**
   - Testing strategy (UI, responsiveness, accessibility, interaction quality).
   - Definition of done and measurable acceptance criteria.

### Quality Expectations

- Keep recommendations practical, implementation-ready, and aligned with current frontend best practices.
- Ensure guidance is specific enough for engineering execution, not just conceptual design direction.
