# Components (`components/`)

Component **specifications** are organized by atomic level in this repo:

| Location                   | Role                                      |
| -------------------------- | ----------------------------------------- |
| `design-system/atoms/`     | Atoms (`.pen` + implementation notes)     |
| `design-system/molecules/` | Molecules (`.pen` + implementation notes) |

Shared **React** implementations live in **`packages/ui`** (package `@agenticverdict/ui`). Start from `/design-system/README.md` for the component index and links to each blueprint.
