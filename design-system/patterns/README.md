# UX patterns (`patterns/`)

Reusable UX and composition patterns are documented alongside the architecture docs:

- **`/docs/architecture/business/design-system/`** — Specifications and standards.
- **`/docs/architecture/business/design-system/generation/`** — Pencil.dev workflows and quick reference.

Atomic compositions (forms, navigation, feedback) are expressed as **molecules** under `design-system/molecules/` (see `../molecules/`).
